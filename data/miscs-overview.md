### Upload Base64 Image and Crop Endpoint[â](#upload-base64-image-and-crop-endpoint "Direct link to Upload Base64 Image and Crop Endpoint")

This endpoint is used to upload an image in Base64 Format and crop it. 

### Delete Image Endpoint[â](#delete-image-endpoint "Direct link to Delete Image Endpoint")

This endpoint is used to delete an image using the image name.

### Get Trained Model List Endpoint[â](#get-trained-model-list-endpoint "Direct link to Get Trained Model List Endpoint")

This endpoint returns an array with the IDs of the models that you have created.

### Get Public Model List Endpoint[â](#get-public-model-list-endpoint "Direct link to Get Public Model List Endpoint")

This endpoint returns a list of all the public models available together with information about their id, status, name, description, etc.

### Clear User's Cache Endpoint[â](#clear-users-cache-endpoint "Direct link to Clear User's Cache Endpoint")

This endpoint is used to clear the user's cache.

### Buy Subscription Plan Endpoint[â](#buy-subscription-plan-endpoint "Direct link to Buy Subscription Plan Endpoint")
